## Do I need this?

You need a Commercial License if **any** of the following are true:

- Your company (of any size, of any revenue) uses qtests, and more than one
  person touches the codebase that uses it.
- You ship qtests as part of a product or service sold to others.
- You run qtests in CI, staging, or production owned by a company.

You do **not** need a Commercial License if you are a single individual
using qtests for your own personal projects, or if you are a genuine
not-for-profit open source project as defined in the `LICENSE` file's
"Open Source Exception".

If you are unsure, the Startup tier is $99/year and removes the question
entirely. We would rather you use qtests with confidence than skip it
because the licensing felt ambiguous.
